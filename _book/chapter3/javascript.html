闭包

什么是闭包？

函数嵌套函数，内部函数可以引用外部函数的参数和变量，参数和变量不会被垃圾回收机制收回。

<!DOCTYPE html> 
<html>
<head> 
<meta charset="utf-8"> 
<script type="text/javascript">
    function aa(c) {
        var a = 12;
        function bb() {
            alert(a);
            alert(c);
        }
        return bb;
    }
    cc = aa(24);
    cc();
    dd = aa(38);
    dd();
</script>
</head>
<body> 
</body>
</html>

修改为封闭函数形式

<!DOCTYPE html> 
<html>
<head> 
<meta charset="utf-8"> 
<script type="text/javascript">
    cc = (function aa(c) {
        var a = 12;
        function bb() {
            alert(a);
            alert(c);
        }
        return bb;
    })(24);
    cc();
