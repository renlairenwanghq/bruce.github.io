## ubuntu16 安装sublime-text2

####1. 安装

通过apt-get install来安装，可以如下来做：

将sublime-text-2添加到软件源中．

sudo add-apt-repository ppa:webupd8team/sublime-text-2

更新软件

sudo apt-get update

安装

sudo apt-get install sublime-text

安装成功，就可以在dash中找到sublime-text-2．

#### 2.安装package Control

View --> Show Console 

按照官网`https://packagecontrol.io/installation#st2`描述选择代码，粘贴到console。
