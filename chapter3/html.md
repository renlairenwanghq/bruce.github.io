## HTML

### 基本结构

```
<!DOCTYPE html> //文档声明
<html>
<head> //头部
<meta charset="utf-8"> //编码声明
</head>

<body> //内容体
</body>

</html>
```



#### 文档类型

* xhtml 1.0
* html5



**alt 属性**



**注释**

```
<!--注释 -->
```
